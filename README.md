## Descripción
Web promocional de un videojuego que incluye una landing page
y una demo interactiva de pantallas de configuración del juego.

## Tecnologías
- HTML5
- CSS3
- JavaScript (jQuery)
- Bootstrap 5.3 
- API externa para datos dinamicos
